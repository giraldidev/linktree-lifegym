# 🏋️ LifeGym - Linktree Profissional

![Banner LifeGym](preview.png)

## 📋 Sobre o Projeto

Este é um projeto de **Linktree personalizado** desenvolvido como um trabalho freelance para a **Academia LifeGym**. O objetivo principal foi criar uma central de links (hub) que reflete a identidade visual premium da marca, facilitando o acesso dos alunos e interessados às principais plataformas da academia.

O design foi concebido com uma estética **Dark & Gold**, utilizando tons de preto profundo e dourado vibrante para transmitir força, autoridade e exclusividade.

---

## ✨ Funcionalidades

- 📱 **Design Totalmente Responsivo**: Otimizado para smartphones, tablets e desktops.
- 🕒 **Seção de Horários**: Visualização clara e elegante do horário de funcionamento.
- ⚡ **Performance Ultra-Rápida**: Carregamento instantâneo sem dependências pesadas.
- 🎨 **Identidade Visual Premium**: Cores e tipografia (Montserrat) alinhadas ao branding da LifeGym.
- 🔗 **Botões de Call-to-Action**: Links diretos para Matrícula (WhatsApp), Wellhub, Redes Sociais e Localização.

---

## 🛠️ Tecnologias Utilizadas

Para garantir leveza e facilidade de manutenção, o projeto foi construído puramente com:

- **HTML5**: Estrutura semântica e SEO.
- **CSS3 (Vanilla)**: Design moderno, Flexbox, Variáveis CSS e animações customizadas.
- **Font Awesome**: Ícones vetoriais de alta qualidade.
- **Google Fonts**: Tipografia Montserrat para um visual profissional.

## 📸 Prévia do Design

> **Nota**: Para que a imagem apareça no README, salve o print da tela como `preview.png` na pasta raiz do projeto.
